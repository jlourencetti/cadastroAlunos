> CADASTRO ALUNOS

## Requisitos

- Python 3+
- SQLite3
- Django


# Como fazer a consulta dos alunos

- python manage.py runserver

```
http://127.0.0.1:8000/
```

# Acesso ao admin para excluir, editar, adicionar 

- Criar usuario
python manage.py createsuperuser

```
http://127.0.0.1:8000/admin/
```
